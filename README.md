# NAO-interactive-language-teacher-project

Developed an interactive language learning system using the NAO humanoid robot. The project aims to enhance language learning through an engaging and interactive experience that combines speech, movement, gestures, and visual display.

The system allows users to select a language, proficiency level, and preferred daily learning duration. The learning process is divided into four main modules: Listening, Speaking, Reading, and Writing.

During the listening phase, NAO plays audio clips followed by comprehension questions, supported by physical gestures to enhance engagement. In speaking, the robot assists users in pronunciation and provides feedback through voice and movement. The reading module displays text on NAO’s screen with interactive questions, while the writing module involves sentence structuring activities supported by visual cues.

The robot enhances the learning experience by using gestures such as clapping, nodding, and repeating instructions in an encouraging manner, creating a more engaging and motivating environment.

This project integrates robotics, artificial intelligence, and human-computer interaction to create an innovative and interactive language learning system.
